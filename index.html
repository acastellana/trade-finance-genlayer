The Boundary
A normal oracle can
Lock a benchmark rate and return a number.
A normal oracle cannot
Resolve which shipment record controls when evidence conflicts.
GenLayer
Evaluates the evidence, returns a machine-usable verdict, and lets the same contract continue.
Late payment changes settlement timing — roll cost, FX exposure, interest.
Late shipment changes whether and how much of the escrowed funds should be released.
This demo covers the shipment condition. The contract pre-defines the remedy; GenLayer decides the disputed fact.
Trade Lifecycle

From invoice to settlement

Five steps. One contract. If shipment timing is contested, GenLayer classifies the delay. The contract applies the pre-agreed remedy — from full release to proportional adjustment to cancellation.

1
Invoice Created
150,000 BOB trade registered on-chain.
2
Benchmark Locked
GenLayer validators agree on 0.493 PEN/BOB.
3
Settlement Funded
150,000 × 0.493 = 73,950 PEN. Importer pre-funds conditional escrow. Release depends on shipment status.
4
Shipment Checked
If uncontested, full settlement is released. If contested, GenLayer classifies the delay. The contract applies the pre-agreed remedy.
